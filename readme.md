## Startroupe

Site based on [Laravel Framework](http://laravel.com/docs).

###To get started
install composer and nodejs
<pre><code>
git clone https://github.com/bharatbgarg4/startroupe.git
cd startroupe
composer install
npm install
npm install -g gulp
gulp && gulp watch
php -S localhost:8000 -t public/
</code></pre>
*Create .env file*
